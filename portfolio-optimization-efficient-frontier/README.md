# Portfolio Optimization using Efficient Frontier

## Overview
This project applies Modern Portfolio Theory to construct optimal portfolios by analyzing the trade-off between risk and return.

## Objective
Identify the optimal asset allocation that maximizes returns for a given level of risk using simulation and optimization techniques.

## Methods
- Monte Carlo simulation of thousands of portfolio weight combinations
- Calculation of expected return, volatility, and Sharpe ratio
- Visualization of the Efficient Frontier

## Key Results
- Identified optimal portfolios with the highest Sharpe ratio
- Demonstrated the risk-return tradeoff across different portfolio allocations

## Tech Stack
Python, NumPy, Pandas, Matplotlib

## Key Skills
Portfolio optimization, simulation, financial modeling, data visualization
